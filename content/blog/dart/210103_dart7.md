# 산술연산자
`+`, `-`, `*`, `/`, `~/`, `%`, `++`, `--`  

`/` : 나눗셈  
`~/` : 몫. 정수를 결괏값으로 가짐.  
`%` : 나머지  

# 할당 연산자
`=`, `+=`, `-=`, `*=`, `/=`, `~=`  

# 관계 연산자(비교 연산자)
`==`, `~=`, `>`, `<`, `>=`, `<=`, `!=`  



# 타입 검사 연산자
`as` : 형 변환  
`is` : 객체가 특정 타입이면 true  
`is!` : 객체가 특정 타입이면 false (=특정 타입이 아니면 true)
